# NeurVana Website

A modern, responsive website for NeurVana neurosomatic therapy studio.

## Tech Stack
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Deployment

This site is deployed on Netlify.
